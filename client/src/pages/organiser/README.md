# Organiser Pages

This folder contains organiser-facing pages for managing events.

---

## 📄 Pages

### Organiser Home (`OrganiserHome.tsx`)

* Dashboard for event organisers
* Displays:

  * Event statistics (active events, attendees, engagement)
  * List of organiser's events
* Includes quick action section for event management

---

## 🧩 Components Used

* Sidebar
* Topbar
* Footer
* EventGrid
* StatCard

---

## 📌 Notes

* Some actions (e.g., Create Event, Manage Event) are placeholders
* These will be connected to backend functionality in future development
* Page is designed to support event CRUD operations later
