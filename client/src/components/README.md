# Components

This folder contains reusable UI components used across the application.

## Structure

- layout/ → Sidebar, Topbar, Footer
- events/ → EventCard, EventGrid
- filters/ → SearchBar, FilterBar
- ui/ → Tabs, StatCard, and other small reusable UI pieces
- qr/ → QR code related components

## Purpose

To avoid repeating code across pages and keep the UI consistent.
